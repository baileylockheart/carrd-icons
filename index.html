Explanation

I want to be able to use carrd icons in a way that works similarly to fontawesome, with a simple copy-paste of a short link and a short code for each icon. I know how to do something similar if you have the file locally, which you can check out in the Alternative tab! Still though, I want to be able to link to it more easily, without downloads or taking up a lot of space inline!

Alternative

1. Check the icons file to make sure that it has the icons you want, since I'm still adding to it, and download it.

2. Make sure to put the icons file in the base directory/main folder of your website.

3. Next, paste this between the header tags of your website.

4. Then, put this at the very end of your website. It should be the last thing before the closing body tag.

For each icon, you'll use this format, changing "name" out for whatever the name of the icon is!

You'll have to repeat steps 3-4 on each page with icons, but generally, this works as far as I've tried it! My only big issue with this method is that you have to download it, rather than linking to it without said download. I know that might be a plus for some, but I'm aiming for convenience :'D
